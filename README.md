# 智能笔记研究助手

一个基于 Ollama 本地大模型（qwen3.5:9b）的 RAG 智能体项目。用户输入自然语言问题，智能体会自动检索本地笔记目录，提取相关片段，并生成带引用来源的回答。

## 核心特性

- 真实大模型接入（Ollama OpenAI 兼容接口）
- 智能体工作流（规划 → 工具调用 → 增强生成 → 日志输出）
- 检索增强生成（RAG），支持关键词匹配检索
- 安全护栏（输入校验 + 路径限制 + 人工确认）
- 自动化测试（8 个 pytest 用例 100% 通过）
- 评估样本集（5 个场景 100% 通过）
- 成本统计（每次调用记录 token 用量）
- Conda 环境一键复现
